# BEXA.LONDON

My personal link hub and about page — a little retro pixel OS on the web.

Plain HTML, CSS, and JavaScript. No build step, no framework, no dependencies.

## What it is

**[BEXA.LONDON](https://bexa.london)** is where I point people to everything I do online. It looks like a vintage desktop: floating windows, a taskbar, and a START menu.

| Page | What you'll find |
|------|------------------|
| `index.html` | Main links — projects, communities, and places to find me |
| `about.html` | A bit about me — profile, stats, bio, and what I'm working on |

Both pages share the same starry background, window chrome, and bottom taskbar with social links.

## What's in here

```
index.html      Home / link hub
about.html      About me
styles.css      Shared styles
script.js       Stars, START menu, window controls
images/me.png   Profile photo
```

## The vibe

- Retro OS windows with minimize, maximize, and close
- Pixel fonts: [VT323](https://fonts.google.com/specimen/VT323) and [Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P)
- Taskbar with social icons and a Windows-style START menu (desktop)
- Mobile-friendly layout with a simplified taskbar on small screens
